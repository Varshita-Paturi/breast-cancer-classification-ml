# Breast Cancer Classification using Machine Learning

## Overview
This project uses Logistic Regression to classify tumors as
malignant or benign using the Wisconsin Breast Cancer Dataset.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Google Colab

## Workflow
1. Data Loading
2. Data Exploration
3. Train-Test Split
4. Logistic Regression Model
5. Model Evaluation
6. Prediction System

## Results
Training Accuracy: 94.9%
Testing Accuracy: 93.0%

## Dataset
Wisconsin Breast Cancer Dataset
